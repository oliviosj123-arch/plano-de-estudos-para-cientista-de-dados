Plano de Estudos – Cientista de Dados

Visão do Dia a Dia
- Coletar e organizar grandes volumes de dados
- Explorar e analisar dados para encontrar padrões
- Criar modelos estatísticos e de machine learning
- Comunicar insights para áreas de negócio
- Trabalhar com ferramentas de programação e visualização

Mapa de Skills
Core Skills (essenciais):
- Programação em Python
- Estatística e Probabilidade
- Manipulação e análise de dados (Pandas, NumPy)

Nice-to-Have (complementares):
- Machine Learning (Scikit-learn)
- Visualização de dados (Matplotlib, Seaborn, Power BI)

Ferramentas e Tecnologias:
- Python
- SQL
- Jupyter Notebook

Roadmap de 90 Dias (12h/semana)
Mês 1 – Fundamentos
Semana 1-2: Instalar ambiente Python/Jupyter, lógica básica de programação
Semana 3-4: Estatística descritiva, manipulação de dados com Pandas e NumPy

Mês 2 – Prática
Semana 5-6: Exploração de datasets reais, visualização com Matplotlib/Seaborn
Semana 7-8: Introdução ao Machine Learning com Scikit-learn, regressão e classificação

Mês 3 – Portfólio e Preparação
Semana 9-10: Projeto de análise exploratória completo
Semana 11-12: Publicar no GitHub e preparar apresentação

Projeto de Portfólio
Projeto: Análise de Dados de Vendas
O que fazer: Explorar dataset de vendas, identificar padrões e prever tendências simples
Entregáveis: Notebook com análise, visualizações claras, modelo preditivo básico
Critérios de Aceitação: Código limpo, visualizações interpretáveis, conclusões estruturadas
Dica: Use datasets do Kaggle ou da DIO

Roteiro de Entrevistas
1. O que é um DataFrame no Pandas? → Estrutura tabular similar a planilha para manipulação de dados
2. Diferença entre regressão e classificação? → Regressão prevê valores contínuos, classificação prevê categorias
3. Como lidar com dados faltantes? → Remoção, imputação por média/mediana ou modelos
4. Explique um projeto de análise que você fez → Estruture em problema → dados → técnicas → resultados
5. O que é overfitting? → Modelo aprende demais os dados de treino e não generaliza
